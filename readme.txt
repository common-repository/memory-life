== Description ==

A plugin to view your saved memories in your blog. It displays your photos, video, sounds, texts from your memory-life.com account.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download the plugin and unzip it to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. change parameters title, token (token is given when you share an album in memory-life.com)
